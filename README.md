# Banking Transaction Analyzer

A Python tool that scans financial transaction datasets and flags potentially suspicious activity using simple rule-based detection.

## Features

* Reads transactions from a CSV file
* Flags large transfers, repeated rapid transactions, and unusual country changes
* Outputs a summary of suspicious transactions
* Modular programming used by separating parsing, logic, and reporting

## Tech Stack

Python, CSV processing, basic data analysis

## How to Run

Run main.py

## Purpose

Built to practice structured data processing, modular program design, and applying software to financial monitoring problems.
